# Archived Configs

This folder stores historical or experiment-specific YAML files that are no longer active defaults.

Active configs live in:
- `config_files/base.yaml`
- `config_files/presets/lite.yaml`
- `config_files/presets/full.yaml`
- `config_files/presets/prod.yaml`

Use CLI overrides via `scripts/pipeline.py` instead of creating per-run config copies.
